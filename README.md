# Social_network

В этом пет-проекте можно просматривать посты. С помощью django-allauth реализована регистрация через гугл. Можно оставлять свои посты, ставить лайки, оставлять комментарии, добавлять в друзья других пользователей. Проект разделён на две части: на приложение для работы с пользователями (Profiles), и на приложение для работы с постами (Posts)
Был создан с целью углубления понимания аутентификации пользователей и работы Django (в частности работа context_processors, signals, models.Manager, работа со слагами).
